======================
**Developer Guide**
======================

This guide is intended to assist software developers who want to develop applications by
using the Rackspace Cloud Orchestration service REST application programming interface (API).
It fully documents the Cloud Orchestration API, which enables developers to interact with the
components of the service.

To use the information provided here, you should have a general understanding of the
Orchestration service. You should also be familiar with the following technologies:

-  *RESTful* web services

-  *HTTP*/1.1

-  JSON data serialization format
